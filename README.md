# Outpost Design Clone

Повна копія сайту outpost.design з усіма анімаціями, 3D-об'єктами та інтерактивними елементами.

## Особливості

- ✅ Повні scroll-анімації з використанням GSAP ScrollTrigger
- ✅ 3D-об'єкти на основі Three.js
- ✅ Hover-ефекти на всіх інтерактивних елементах
- ✅ Паралакс-ефекти при скролі
- ✅ Адаптивний дизайн для всіх пристроїв
- ✅ Плавні переходи та анімації
- ✅ Кастомний курсор
- ✅ Інтерактивні 3D-сцени в hero та work секціях

## Технології

- HTML5
- CSS3 (з анімаціями та transitions)
- JavaScript (Vanilla JS)
- Three.js (для 3D-елементів)
- GSAP (для scroll-анімацій)

## Встановлення та запуск

### Простий спосіб (без встановлення)

Просто відкрийте `index.html` у браузері. Всі залежності підключені через CDN.

### З локальним сервером

1. Встановіть Node.js (якщо ще не встановлено)
2. Запустіть один з наступних команд:

```bash
# Варіант 1: Використовуючи npx serve
npm start

# Варіант 2: Використовуючи live-server
npm run dev
```

Сайт буде доступний за адресою `http://localhost:3000` (або інший порт, зазначений у консолі)

## Налаштування

### Зміна кольорів

Відредагуйте CSS-змінні в `styles.css`:

```css
:root {
    --primary-color: #000000;
    --secondary-color: #ffffff;
    --accent-color: #ff6b6b;
    --text-color: #333333;
    --bg-color: #f8f8f8;
}
```

### Додавання власних зображень

Замініть URL зображень у `index.html` на свої:

```html
<img src="ваше-зображення.jpg" alt="Опис">
```

### Налаштування 3D-об'єктів

Відредагуйте функції `initHero3D()` та `initWork3D()` у `script.js` для зміни геометрії, матеріалів та анімацій.
